================================================================
  QUL Loader Navigation POC  ―  フロー図アニメーション
================================================================

このフォルダのアニメーションをブラウザで見るには:

  ┌─────────────────────────────────────────────────┐
  │                                                 │
  │  【Windows の場合】                              │
  │     start_app.bat  をダブルクリック             │
  │                                                 │
  │  【Mac / Linux の場合】                          │
  │     ターミナルで  ./scripts/start_app.sh        │
  │                                                 │
  └─────────────────────────────────────────────────┘

  → 自動的にブラウザが開いて、アニメが再生できます。


【後片付けについて】
  ブラウザのタブを閉じると、裏で動いていたプログラム
  (Web サーバー) も自動で終了します。
  特別な操作は不要です。


【必要なもの】
  Python 3 がインストールされていること
  (Qt Creator が動く環境なら大抵入っているはずです)


【このフォルダの中身】
  start_app.bat   Windows 用の起動ボタン (ダブルクリック)
  app/            アニメーション本体 (HTML/CSS/JS)。普段は触らない。
  scripts/        起動用スクリプト類 (Mac/Linux の .sh、Pyth
